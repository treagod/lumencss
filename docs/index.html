Principles

CSS only

No build tools required. Just link the stylesheet and go.

Readable tokens

HSL variables for color, spacing, radius, shadows, and type scales.

Solid defaults

Responsive layout, accessible focus, and sensible component states.

Rounded & friendly

Softer radii and subtle surfaces without sacrificing contrast.

Quick include

<link rel="stylesheet" href="lumencss.css">

Use a relative path inside /docs so it works on GitHub Pages project sites.
