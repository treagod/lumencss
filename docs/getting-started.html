Install

Copy the compiled CSS into /docs and link it:

<link rel="stylesheet" href="lumencss.css">

Source tokens live in your repository; no preprocessor is required.

Layout shell

<div class="wrapper">
  <h1 class="is-h1">Hello</h1>
  <div class="card stack">
    <div class="field">
      <label for="email">Email</label>
      <input class="input" id="email" type="email" placeholder="you@example.com" required>
    </div>
    <button class="btn">Submit</button>
  </div>
</div>

Customize tokens

Adjust HSL color scales, spacing, and radii in your theme file. Changes cascade across components automatically.
